# NS Visual

默认只写提示词；用户明确要求生成图片时再调用图像生成能力。

## 读取

- `NOVEL.md`（存在时读取；不要自行创建或修改）
- `art.yaml`
- `memory.yaml`
- 当前章节 `## 正文`
- 指定模型格式时读取 [prompt-patterns.md](references/prompt-patterns.md)

## 输出

- 默认给中英文提示词；指定模型时按目标格式写。
- 角色/场景复用时拆稳定要素和可变要素。
- 完整提示词写 `visuals/*.md`；稳定要素和路径摘要到 `art.yaml`。
- 新视觉禁忌或全局偏好交给 `$ns-canon`。
- 按封面、角色、场景、道具或章节拆分 visuals 文件。
